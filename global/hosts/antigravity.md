# Host: Antigravity

Antigravity does **not** auto-load `AGENTS.md` like Cursor. Use **Customizations** and **Workflows** (each has **Global** and **Workspace** scopes).

## Recommended mapping

- **Workspace Customizations:** Paste the OverkillOS boot obligations: global `~/.overkill/memory/QUICKSTART.md` (Step 0), then `.overkill/identity/SOUL.md`, `.overkill/memory/QUICKSTART.md`, daily logs — align with `AGENTS.md` Step 2.
- **Workspace Workflows:** Add a workflow (e.g. `overkill-boot`) that runs before coding: confirm PLANNING → EXECUTION → VERIFICATION alignment with `.overkill/verification/` gates.
- **Global Customizations:** Optional one-liner culture rules (non-negotiables) shared across workspaces.

## Constraints

- Custom **MCP** connections may be limited; prefer **rules + workflows** + repo files.
- Keep host notes **short**; canonical truth stays in `.overkill/`.
